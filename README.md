# Happy Birthday, Anam

A premium, cinematic full-screen photo reveal section for a romantic birthday landing page.

## Tech

- **React** (functional components)
- **Framer Motion** – text reveal and scroll indicator
- **GSAP** – background zoom (scale 1 → 1.08 over 8s)

## Run

```bash
npm install
npm run dev
```

Open the URL shown in the terminal (e.g. http://localhost:5173).

## Customise

- **Background image:** Edit `hero__bg` in `src/components/HeroSection.css` and set `background-image` to your image URL, or place an image in `public/` and use `/your-image.jpg`.
- **Copy:** Edit the heading, name, and lines in `src/components/HeroSection.jsx`.

## Build

```bash
npm run build
```

Output is in `dist/`.
